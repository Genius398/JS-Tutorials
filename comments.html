Lesson 2 - Comments




If you already know html and css then you will know what is comments. You can use comments for explaining

what this code is for. If you are working with someone then you can use comments to tell them what they

should do next. And people who see the website can't see this comment. Let's check how to do that. Type a

paragraph which you want to make it a comment. Before you do that you need a JavaScript file. .js is the

extension of the JavaScript. You should make it as : TheNameYouWant.js

Type the sentence you want to make it a comment type // + space before it starts or click ctrl+/ .
